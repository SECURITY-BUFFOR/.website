User management gives administrators one place to provision accounts, organize teams, assign permissions, and control access.

## Feature Highlights

- Search existing users and create accounts individually or by CSV import.
- Manage display names, usernames, and email addresses.
- Assign administrative, management, pentester, template editor, client, user, and guest roles.
- Organize users into groups for reusable team membership.
- Search groups and add or remove individual members.
- See whether an account is active and when it was created.
- Force logout, disable, or delete an account when access must be removed.
- Reset user passwords from the administration interface.
