The finding library helps teams reuse approved content instead of rewriting the same security issues from scratch.

## Feature Highlights

- Store reusable findings with severity, CVSS, CWE, tags, and references.
- Prepare standard descriptions, impact statements, recommendations, and proof-of-concept sections.
- Support multiple languages for teams serving different client markets.
- Track translation readiness so content is not used before it is reviewed.
- Keep recurring issues consistent across penetration tests, vulnerability assessments, and red team reports.
