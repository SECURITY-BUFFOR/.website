Account security combines stronger authentication with visibility into suspicious or unexpected sign-in activity.

## Feature Highlights

- Add a second authentication factor to individual user accounts.
- Use a standard authenticator app for verification.
- Set up two-factor authentication directly from the user profile.
- Clearly show whether two-factor authentication is enabled.
- Notify users when the same account is signed in on multiple devices.
- Help users identify concurrent sessions that may require review.
