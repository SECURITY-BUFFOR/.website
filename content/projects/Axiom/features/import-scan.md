Scanner imports reduce manual re-entry by bringing existing security-tool results into the assessment workflow.

## Feature Highlights

- Import results from Nessus, Burp Suite, OpenVAS, OWASP ZAP, Nmap, and Nuclei.
- Accept tool-native XML, JSON, JSONL, and Nessus files.
- Support generic JSON and CSV for results from other sources.
- Upload files by drag and drop or through the file browser.
- Preview parsed findings before adding them to the project.
