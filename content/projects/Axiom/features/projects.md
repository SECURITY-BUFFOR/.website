Assessment projects keep technical work, review, and reporting together from the first finding through final delivery.

## Feature Highlights

- Search and filter projects by status, client, and assessment type.
- Track each engagement through draft, active work, QA review, publication, and archival stages.
- Document findings with severity, CVSS, CWE, affected components, status, tags, and structured technical content.
- Complete reusable report fields with side-by-side Markdown previews and design variables.
- Organize project notes in a hierarchy and preview formatted content while writing.
- Assign individual users and groups with clear project roles.
- Generate, preview, download, and revisit rendered PDF reports.
