Security notifications give users clear visibility into account activity and the tools to respond when a session looks unfamiliar.

## Feature Highlights

- Notify users when a new device signs in to their account.
- Keep recent security alerts available in a central notification panel.
- Show the operating system and browser associated with each session.
- Record when sessions started, ended, and were last active.
- Clearly identify the current active session.
- Let users revoke active sessions they no longer trust.
