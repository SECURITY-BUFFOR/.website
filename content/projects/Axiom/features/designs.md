Report designs turn assessment data into polished, repeatable deliverables that match each team's methodology and visual identity.

## Feature Highlights

- Build reusable report structures from cover pages, content sections, findings, appendices, headers, and footers.
- Define custom report and finding fields for the data required by each assessment methodology.
- Control typography, colors, page formatting, and component styling with HTML and CSS.
- Organize fields into clear categories and mark essential information as required.
- Preview the generated PDF with mock data and catch missing or incorrectly referenced fields before delivery.
- Import, export, version, and localize designs for reuse across teams and client requirements.
